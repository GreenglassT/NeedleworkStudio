About the estimate

  • Efficiency accounts for thread waste from tails, mistakes, and back stitching style
  • Higher fabric counts use less thread per stitch (smaller squares)
  • Standard skeins (DMC, Anchor) are 8.7 yards (8 meters)
  • Always buy at least one extra skein for safety

How it works

  • Fabric count = stitches per inch (e.g., 14-count Aida has 14 stitches per inch)
  • Design size = stitch count ÷ fabric count
  • Suggested fabric size = design size + 6″ (3″ margin on each side for framing)
