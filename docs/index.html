Features

Everything you need for cross-stitch, in one self-contained app.

Thread Inventory

Browse 744 DMC and 400+ Anchor threads. Search by number or name, filter by brand and category, mark ownership status, and track skein quantities.

Pattern Design

Convert images to patterns or design from scratch. 11 drawing tools, 6 stitch types, mirror modes, 50-level undo/redo, and brush size control.

Pattern Viewer

Chart mode with symbol overlays, thread mode with realistic rendering. Zen mode, resizable legend, per-color progress tracking, and 10%-400% zoom.

Import & Export

Import from PDF, JSON, and OXS formats. Export to PDF with cover sheets, SVG vector charts, and standard interchange formats.

Project Management

Track pattern status and per-color completion. Saved patterns gallery with search, sort, batch actions, forking, and materials calculator.

Desktop Sync

Bidirectional sync between the desktop app and your self-hosted server. Patterns, inventory, and thread status stay in sync across devices.

# Create docker-compose.yml and run:
docker compose up -d
docker compose exec needlework-studio python manage_users.py create
